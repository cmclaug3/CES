Help

How to use this site

This site was designed to make dealing with Timesheets, and all that come with them, easier and more convenient.

Things such as viewing Jobs, adding Employees to Timesheets, signing Timesheets, viewing Jobs/hours worked, etc

How to create a Timesheet

The easiest way is to go to the Job section, click on a Job name, then add a Work Day

Physical Timesheets can be added through a specific Work Day for a specific Job

The reason for this is sometimes there can be multiple Timesheets for a given day

For example: working DT or Rivers. The drive there, the work, and the drive back were all different Timesheets and required individual signatures

From the Work Day, which will act as all work on a given day for a given Job no matter how many signatures are required, you can then add a Timesheet

Dealing with Timesheets

Timesheets are composed of multiple Employee Work forms, which represent a row on the Timesheet

For example, an Employees Name, Signature, Time Started, Lunch, Time ended, and if injured

Add as many Employees as needed and fill out accordingly

Remember, the Pin you create will act as your signature For these online forms.
